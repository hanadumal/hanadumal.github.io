我的信息收藏现状

在知乎的使用过程中，我只是把收藏夹当作了一个储物箱，或者垃圾桶。垃圾桶的类比与这个过程很相似的一点：随手收藏（扔垃圾）到垃圾桶之后，你就再也不回去复习（翻垃圾桶)。那么其实你认为对自己未来很有帮助的信息，要像宝藏一样珍藏起来的东西，很可能本质上对你只是垃圾。慢慢的，你的储物箱，甚至你的家里堆满了各种垃圾，无处立脚，恶性循环，越来越懒得处理。

如果只是’收藏’、’以后在看’，其实等同’再也不看’，’我要收破烂’。那这些年我到底收藏了多少无用信息。

在生活中读书也是如此，只买书，不看书，不思考。书本堆积如山，理财的、健身的、计算机的、美术的、传记的。到底什么是你生活的中心点？什么是你生活的重心。

这个问题又可以引申出，到底什么是你的核心竞争力？

‘如果吴亦凡是《三体》中的面壁者，故事情节会怎样发展？’

‘如何评价《爱情公寓》电影版？’

‘如何评价范冰冰的演技？’

‘有哪些看似很「歪」实则很「正」的大实话？’

也许这些内容跟生活有这或多或少的联系，但是健身、护肤、异域风情的文化，并不在’提升职场竞争力的核心中。

一共是 168 + 370 + 138 + 215 = 891条信息，假设每个只要1分钟，我也要15个小时才能读完。时间黑洞就这么产生了。

我可能不会移民，我可能不知道很多华丽的词藻，我可能不懂字体设计，我不知道树懒怎么拉屎。人的精力是有限的，聚焦比广撒网更能获得更好的收益。

但是我真的需要学好语言。不过在如此之多的收藏中，你如何能快速找到你需要的信息？

https://www.zhihu.com/collection/308697616

最多的一个收藏夹中有1300条内容，如何能看的完？聚焦聚焦如此重要。

我可能不知道如何给女生拍照，不过没关系，我已经结婚了。
